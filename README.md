# keuangan_umkm

# 1. Clone Repository

```bash
git clone https://github.com/lisajfry/keuangan_umkm.git
cd keuangan_umkm
```

# 2. Instalasi & Menjalankan frontend_admin
Masuk ke folder admin:
```bash
cd frontend_admin
```

## 2.1 Install dependensi
```bash
npm install
```

## 2.2 Jalankan server admin
```bash
npm run dev
```

# 3. Instalasi & Menjalankan frontend_umkm
Masuk ke folder umkm:
```bash
cd frontend_umkm
```

## 2.1 Install dependensi
```bash
npm install
```

## 3.1 Jalankan server umkm
```bash
npm run dev
```
