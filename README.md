# keuangan_umkm

# Requirement

Node.js versi 10.8.2     
NPM versi 10.8.2

# Dependencies

react             ^19.1.1  
react-dom         ^19.1.1  
react-router-dom  ^7.9.3   
axios             ^1.12.2  
lucide-react      ^0.544.0      
react-icons       ^5.5.0   
recharts          ^3.2.1   


# 1. Clone Repository

```bash
git clone https://github.com/lisajfry/keuangan_umkm.git
cd keuangan_umkm
```

# 2. Instalasi & Menjalankan frontend_admin
Masuk ke folder admin:
```bash
cd frontend_admin
```

## 2.1 Install dependensi
menginstall semua package otomatis
```bash
npm install
```

## 2.2 Jalankan server admin
```bash
npm run dev
```

# 3. Instalasi & Menjalankan frontend_umkm
Masuk ke folder umkm:
```bash
cd frontend_umkm
```

## 2.1 Install dependensi
```bash
npm install
```

## 3.1 Jalankan server umkm
```bash
npm run dev
```
